# Python Flask微博舆情分析系统 python143

## 项目概述

本项目是一款基于Python Flask框架开发的微博舆情分析系统，集成了丰富的可视化工具，便于用户直观了解舆情动态。

## 技术栈

- 前端：HTML、CSS、JavaScript
- 后端：Python Flask
- 数据库：MySQL
- 可视化：ECharts

## 功能模块

### 文章分析

- 提取文章关键信息，分析文章情感倾向

### IP分析

- 分析IP地址，识别地域分布

### 评论分析

- 对评论内容进行情感分析，获取用户观点

### 舆情分析

- 整合文章、评论、IP等多方面信息，形成舆情报告
- 带图表展示，直观反映舆情趋势

## 系统角色

- 管理员：负责系统维护、数据监控等
- 普通用户：进行舆情查询、分析等

## 运行环境

- 操作系统：Linux或Windows
- Python版本：3.x
- 数据库：MySQL 5.x

## 部署步骤

1. 安装Python及相关依赖
2. 安装MySQL数据库
3. 导入数据库初始化数据
4. 修改配置文件，配置相关参数
5. 运行Flask应用

## 目录结构

```
- /app

  - /views

```
